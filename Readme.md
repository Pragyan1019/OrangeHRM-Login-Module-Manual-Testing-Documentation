# OrangeHRM Login Module — Manual Testing Project

## Project Overview
Manual testing of OrangeHRM login functionality
covering functional, security, and regression testing.

## Tools Used
- Jira + Xray (Test Management)
- Apache JMeter (Performance Testing)
- GitHub (Documentation)

## Test Coverage

- Functional, Smoke, Security Testing
- SQL Injection validation
- Session/Logout testing
